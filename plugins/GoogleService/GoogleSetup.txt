<ol>
    <h3><b>Google Compute has not been configured on this machine yet!</b></h3>
    <li>
        If Google api client is not installed, run <b>pip install --upgrade google-api-python-client</b>
    </li>
    <li>
        Go to the <a href="https://console.cloud.google.com/" target="_blank">google cloud website</a> and create an account or sign onto your existing one.
    </li>
    <li>
        Go to your <a href="https://console.cloud.google.com/cloud-resource-manager" target="_blank">cloud resource manager</a> and select or create a Google Cloud Platform project
    </li>
    <li>
        Make sure that <a href="https://cloud.google.com/billing/docs/how-to/modify-project" target="_blank">billing is enabled</a> for your Google Cloud Platform Project
    </li>
    <li>
        Install the <a href="https://cloud.google.com/sdk/" target="_blank">Cloud SDK</a>
    </li>
    <li>
        Go to your command line and <b>"run gcloud auth application-default login"</b>
    </li>
    <li>
        <a href="https://cloud.google.com/apis/docs/enable-disable-apis?hl=en&ref_topic=6262490&visit_id=636909616876722358-4171110160&rd=1" target="_blank">Enable</a> the Google Cloud Storage API and Cloud Resource Manager API 
    </li>
    <li>
        <a href="https://cloud.google.com/storage/docs/creating-buckets" target="_blank">Create a storage bucket</a> and note the bucket name for later
    </li>
    <li>
        Create a <a href="https://console.cloud.google.com/projectselector/apis/credentials/serviceaccountkey?supportedpurview=project" target="_blank">service account for authentication</a>
        <ol>
            <li>From the Service account list, select <b>New service account.</b></li>
            <li>In the <b>Service account name field</b>, enter a name.</li>
            <li>From the <b>Role list,</b> select <b>Project > Owner.</b></li>
            <li>Click <b>Create.</b> This should download a JSON key</li>
            <li>Move this key file into the <b>googleCredentials</b> folder found in <b>guppi/plugins</b></li>
        </ol>
    </li>
    <li>
        Follow <a href="https://cloud.google.com/compute/docs/instances/adding-removing-ssh-keys#project-wide" target="_blank">these instructions</a>
         to create and add a public SSH key to your project. Save the private SSH key as gc_rsa.pem and move it into the GoogleService file.
    <li>
    <li>
        Go to project metadata and save the public RSA key to SSH keys
    <li>
        Reload the <b>"guppi cloud"</b> command
    </li>
</ol>
